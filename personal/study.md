## PART 6: QUICK-REFERENCE CHEAT SHEET

### Nmap Scanning
```
nmap -sC -sV -p- -T4 <target>           # Full scan with scripts & versions
nmap -sU --top-ports 100 <target>        # Top 100 UDP ports
nmap -sV --script vuln <target>          # Vulnerability scan
nmap -sn 192.168.1.0/24                  # Ping sweep (host discovery)
nmap -p 445 --script smb-vuln* <target>  # SMB vulnerability scan
```
